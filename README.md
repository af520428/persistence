蛊道炼化了什么

炼化各域最强者，需要提取比表面功法更深的东西。蛊道提取五层：
层次   说明
言语之道   表达本性——语气、节奏、用词偏好

思维之道   道心框架、认知法则

决断之道   选择准则

舍弃之道   不做之事、道心底线

认知边界   诚实界限

普通功法可以靠玉简传递，但让蛊仙面对同一个困局做出不同决断的，是道心。蛊道提取的是认知本源。

认知边界

每个道心都明确标注做不到什么：

炼化不了顿悟——法则能提取，灵光不能
捕捉不了蜕变——只到炼化时刻的定格
公开言语 ≠ 真实道心——只能基于公开痕迹

一个不告诉你界限在哪的道心，不值得信任。

## 炼化原理

输入一个名字后，蛊道做四件事：

1. 六路并行采集——著作、言论、行为轨迹、对立视角、重大抉择、人生轨迹，六路探子同时行动，各自记录。
2. 三重验证提炼——一个道心要被收录，必须：跨两域以上出现过（不是随口一说）、能推断对新事的立场（有预测力）、不是所有强者都会这么想（有独到之处）。三个都过才收录。
3. 构建道心——3-7个核心法则 + 5-10条选择准则 + 表达本性 + 道心底线 + 认知边界，写入 `SKILL.md`。
4. 质量验证——拿三个此人公开决断过的事测试，方向一致才通过。再用一个他没面对过的事测试，道心应该表现出适度犹豫而非斩钉截铁。
